Harcsin Function

Returns the inverse hyperbolic sine of a number. The inverse hyperbolic sine is the value whose hyperbolic sine is number, so harcsin(hsin(number)) equals number.

Syntax

harcsin(number)

Number   is any valid numeric expression.

Example

Expression Description (Result)
harcsin(-2.5) Inverse hyperbolic sine of -2.5 (-1.64723)
harcsin(10) Inverse hyperbolic sine of 10 (2.998223)
