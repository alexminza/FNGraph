! Operator

Returns the factorial of a number. The factorial of a number is equal to 1*2*3*...* number.

Syntax

number!

Number   is any valid numeric expression

Remarks

If number is negative, the operator returns an error. If number is not an integer, it is truncated.

Example

Expression Description (Result)
3! Factorial of 3, or 1*2*3 (6)
-3! First te factorial is calculated and then the result is negated (-6)
(-3)! Because the number is negative, an error is returned
1.9! Factorial of the integer of 1.9 (1)
0! Factorial of 0 (1)
