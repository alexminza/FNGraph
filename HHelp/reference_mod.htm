Mod Function

Returns the remainder after number is divided by divisor.

Syntax

mod(number,divisor)

Number   is any valid numeric expression for which you want to find the remainder.

Divisor   is any valid numeric expression by which you want to divide number.

Remarks

If divisor is 0, mod returns an error.

Example

Expression Description (Result)
mod(3, 2) Remainder of 3/2 (1)
mod(12.6, 5) Remainder of 12.6/5. The first argument is rounded to an integer and the remainder is calculated. (3)
