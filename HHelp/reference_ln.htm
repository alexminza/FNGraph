Ln Function

Returns the natural logarithm of a number. Natural logarithms are based on the constant e.

Syntax

ln(number)

Number   is any valid numeric expression for which you want the logarithm.

Remarks

Use the log function to calculate logarithms to any base.

Example

Expression Description (Result)
ln(86) Natural logarithm of 86 (4.4543)
ln(e) Natural logarithm of the value of the constant e (1)
ln(exp(3)) Natural logarithm of e raised to the power of 3 (3)
