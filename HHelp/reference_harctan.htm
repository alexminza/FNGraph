Harctan Function

Returns the inverse hyperbolic tangent of a number. Number must be between -1 and 1 (excluding -1 and 1). The inverse hyperbolic tangent is the value whose hyperbolic tangent is number, so harctan(htan(number)) equals number.

Syntax

harctan(number)

Number   is any valid numeric expression between -1 and 1.

Example

Expression Description (Result)
harctan(0.76159416) Inverse hyperbolic tangent of 0.76159416 (1, approximately)
harctan(-0.1) Inverse hyperbolic tangent of -0.1 (-0.1003)
