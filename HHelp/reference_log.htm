Log Function

Returns the logarithm of a number to the base you specify.

Syntax

log(number,base)

Number   is any valid numeric expression for which you want the logarithm.

Base   is any valid numeric expression representing the base of the logarithm.

Example

Expression Description (Result)
log(8,2) Logarithm of 8 with base 2 (3)
log(86,e) Logarithm of 86 with base e (4.4543)
