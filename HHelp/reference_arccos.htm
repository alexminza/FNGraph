Arccos Function

Returns the arccosine, or inverse cosine, of a number. The arccosine is the angle whose cosine is number. The returned angle is given in radians in the range 0 (zero) to pi.

Syntax

arccos(number)

Number   is any valid numeric expression representing the cosine of the angle you want and must be from -1 to 1.

Remarks

If your argument is in degrees, multiply it by pi/180 or use the radians function to convert it to radians.

Example

Expression Description (Result)
arccos(-0.5) Arccosine of -0.5 in radians, 2*pi/3 (2.0944)
arccos(-0.5)*180/pi Arccosine of -0.5 in degrees (120)
degrees(arccos(-0.5)) Arccosine of -0.5 in degrees (120)
